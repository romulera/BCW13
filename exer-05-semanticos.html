Esse site é feito para esclarecer os elementos semânticos do HTML

MAIN

Essa tag main é a responsavel pelo conteúdo principal, ela é unica em um arquivo HTML.
Ele não pode ser descendente de um ARTICLE, ASIDE, FOOTER, HEADER ou NAV.
Aqui você coloca tudo que é mais importante.



SECTION

Como o proprio nome diz é uma sessão do site. Section ele definie uma seção do seu site, algo secundario, que pode ser usando mais de uma vez no mesmo arquivo HTML.
Exemplo: Sessao receitas doces, filmes de ação...



ARTICLE

A tag article é uma tag que demarca um conteudo importante do site.
Exemplo: Um post de uma receita nova na sessão de receitas doces.



NAV

Vou usar a tag article para falar da tag NAV que é onde estarão os links para navegar entre os principais conteúdos do site.
Nav pode ser utilizado para criar um conjunto de links para navegar entre as receitas por exemplo.



ASIDE

Aside ou ao lado, é uma tag usada para adicionar um conteudo complementar ao conteudo main.



FOOTER

Footer é usado para criar um rodapé, para colocar informações que precisão estar presente, porem não precisa estar em evidencia.
Exemplo: Endereço, direitos autorais, etc...
